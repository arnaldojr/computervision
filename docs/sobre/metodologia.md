# Metodologia

## Abordagem Pedagógica

A disciplina adota uma abordagem teórico-prática com foco em aprendizagem baseada em projetos (Project-Based Learning). 

Os alunos trabalharão em ciclos de desenvolvimento incremental, construindo soluções reais de visão computacional.

## Estratégias de Ensino

- **Aulas expositivas dialogadas**: Apresentação dos conceitos teóricos com interação entre professor e alunos
- **Demonstrações práticas**: Exemplos em tempo real de implementação de algoritmos de visão computacional
- **Implementação incremental de projeto**: Desenvolvimento contínuo de um projeto ao longo da disciplina
- **Estudos de caso reais**: Análise de aplicações reais de visão computacional em diferentes domínios
- **Desenvolvimento de sistema aplicado**: Criação de soluções funcionais para problemas específicos
